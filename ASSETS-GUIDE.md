## ğŸ”§ **Image Optimization**

### Before Upload:
1. **Compress**: Use TinyPNG.com or Squoosh.app
   - Target: JPG 70-85% quality
   - Target file size: <200KB for large images, <100KB for thumbnails

2. **Convert to WebP**: Better compression
   ```bash
   # Using cwebp (install with: npm install -g cwebp-bin)
   cwebp -q 80 input.jpg -o output.webp
   ```

3. **Resize**: Use correct dimensions
   ```bash
   # Using ImageMagick
   magick input.jpg -resize 1200x800 -quality 85 output.jpg
   ```

4. **Add Alt Text**: Every image in HTML needs descriptive alt
